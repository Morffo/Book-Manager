# Book-Manager

## First step: Clone the repository

```Bash
git clone https://github.com/Morffo/Book-Manager
```

## Second step: create and activate virtual environment:

```Bash
python -m venv venv
source venv/bin/activate
```

## Launch:

```Bash
python music_oop.py
```
OR

```Bash
python tkinter_interface.py
```
